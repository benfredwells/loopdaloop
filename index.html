Loopdaloop - on any desktop platform!

Loopdaloop works on Windows, Mac, Linux and ChromeOS - take your pick!

Multiple oscillators, filters, time varying parameter control, synthesizer setup visualization, and a mouse, keyboard and touch piano keyboard are all available in the development version.

The release version is available on the Chrome Web Store. It is regularly updated automatically, so make sure you use it regularly to enjoy the latest goodness!

Designer Templates

Loopdaloop is an open source project, developed by a software engineer as a therapeutic break from his day job. It's built with the HTML5 WebAudio API, which is pretty cool and does all the heavy lifting.

Right now it is a competent synthesizer. One day it will also support sound fonts and looping.
