UI Layout

Contoured value:
  - Flat
    - value
  - Oscillate
    - center value
    - wave type
    - amplitude
    - frequency
  - Sweep
    - type (linear, exponential, approach)
    - start value
    - end value
    - time
  - Thwang
    - sweep start
    - sweep type
    - sweep time
    - center value
    - oscillation type
    - amplitude
    - frequency
  - ADSR
    - initial value
    - attack delay
    - attack value
    - attack time
    - attack hold
    - sustain value
    - decay time
    - release value
    - release time

Oscillator
  - Enabled / Disabled
  - Type (sine, square, triangle, sawtooth[, noise])
  - Octave / Semitone offset [not for noise]
  - Detune (contour)
    - range -5% to +10%
    - controls detune input
  - Volume (contour)

Pitch (contour pitch)
  - range -50% to +100%
  - base oscillator frequency = note frequency * (1 + pitch)

Filter
  - Type (low pass / high pass)
  - Order
  - Frequency (contour ff)
    - range -50% to 800%
    - cut off frequency = note frequency * (1 + ff)
  - Track
  - Resonance (contour res)

Envelope (constrained contour)
  - range 0 to 1
  - initial value 0
  - attack value 1
  - attack delay 0
  - type ADSR
  - release value 0
  - controls output gain
